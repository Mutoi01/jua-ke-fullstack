## Built with and Deployment
* React for the client side.
* Ruby on rails for the server side.

* To set up the backend, run:
* $ bundle install
* $ rails db:migrate db:seed
* $ rails s

* Then, in a new terminal, run the frontend:

* $ npm start --prefix client
